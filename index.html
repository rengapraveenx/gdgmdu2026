Step-by-step

Get Running in Minutes

Follow these steps in order — from install to first AI response.

1

Download & Install Ollama

Ollama runs large language models locally. Pick your OS:

🪟
Windows
ollama.com/download
↗ 🍎
macOS
ollama.com/download
↗
💡 After install, Ollama runs silently in the background. No browser tab needed.
2

Install Python 3.11+

The frontend app is a Python script. Install Python if you haven't already:

🪟
Windows
python.org/downloads
↗ 🍎
macOS (M1–M4)
GeeksforGeeks guide
↗
⚠️ Windows users — check "Add Python to PATH" during installation.
verify install 
python --version
# Expected: Python 3.11.x or higher
3

Pull the Gemma 3 Model

Open a terminal (Command Prompt / Terminal) and pull the Gemma 3 model. This downloads ~3 GB once and caches it locally.

terminal 
ollama pull gemma3
# Downloads model weights — takes a few minutes
💡 Test it works: ollama run gemma3 — type a message, press Enter, then Ctrl+D to exit.
4

Get the App Files

Two options — use the workshop bundle (from the drive download above) or clone directly from GitHub:

terminal 
git clone https://github.com/venketraj/GDG-Ollama-DEMO.git
cd GDG-Ollama-DEMO
📁 If you used the Drive download, just extract the ZIP and cd into that folder.
5

Install Python Dependencies

Inside the project folder, install required packages:

terminal · inside project folder 
pip install -r requirements.txt
6

Run the App 🚀

Make sure Ollama is running (it starts automatically after install). Then launch the frontend:

terminal · inside project folder 
python app.py
🎉 Open http://localhost:8501 (or the URL shown in terminal) in your browser. Start chatting with Gemma 3 — fully offline!

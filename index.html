Step-by-step

Get Running in Minutes

Follow these steps in order — from install to first AI response.

1

Download & Install Ollama

Ollama runs large language models locally. Pick your OS:

🪟
Windows
ollama.com/download
↗ 🍎
macOS
ollama.com/download
↗
💡 After install, Ollama runs silently in the background. No browser tab needed.
2

Install Python 3.11+

The app uses Streamlit. Install Python 3.11 if you haven't already:

🪟
Windows
python.org/downloads
↗ 🍎
macOS (M1–M4)
GeeksforGeeks guide
↗
⚠️ Windows — check "Add python.exe to PATH" before clicking Install.
macOS — install via Homebrew (recommended):
macOS terminal 
brew install python@3.11
verify 
python3 --version   # mac
python  --version   # windows — expected: 3.11.x or higher
3

Pull the Gemma 3 Model

Open terminal and pull the model. Downloads once, cached locally. At the workshop, use USB transfer if no internet — see callout below.

terminal 
ollama pull gemma3:270m
# Verify after download:
ollama list
💡 No internet at venue? Copy .ollama/models/ folder via USB to the same path on the offline machine. Launch Ollama, run ollama list to verify.
📂 macOS: /Users/<you>/.ollama/models/ 📂 Windows: C:\Users\<you>\.ollama\models\
4

Get the App Files

Two options — use the workshop bundle (from the drive download above) or clone directly from GitHub:

terminal 
git clone https://github.com/venketraj/GDG-Ollama-DEMO.git
cd GDG-Ollama-DEMO
📁 If you used the Drive download, just extract the ZIP and cd into that folder.
5

Create Virtual Environment & Install Dependencies

Inside the project folder, create an isolated Python environment then install packages:

macOS terminal 
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
Windows cmd 
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
📦 Installs: streamlit, ollama, requests
6

Run the App 🚀

Make sure Ollama is running in the background and venv is activated. Then launch the Streamlit UI:

terminal · inside project folder 
streamlit run app.py
🎉 Browser opens automatically at http://localhost:8501. Start chatting with Gemma 3 — fully offline!
